About

Local Seriesly is an reimplementation of Seriesly by Stefan Wehrmeyer. It lets you fetch the data of new episodes of your favourite tv show from tvrage.com and displays it as an html document.

Local Seriesly runs without Googles AppEngine or Django. This makes it easy to use on any computer or webserver with Python installed. To keep it simple, most of the features of the original Seriesly, like RSS and XMPP support, will not be implemented.

You want to contribute?

Since Local Seriesly is Open Source anyone can use and improve the project. The code is maintained at this GitHub-Repository
